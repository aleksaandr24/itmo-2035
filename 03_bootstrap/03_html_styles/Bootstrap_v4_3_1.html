Twitter Bootstrap
Свободный набор инструментов для создания сайтов и веб-приложений

Включает в себя HTML и CSS шаблоны оформления для типографики, веб-форм, , меток,блоков, навигации и прочих компонентов веб-интерфейсов, включая JavaScript расширения.
Сетки — заранее заданные размеры колонок, которые можно сразу же использовать, например, ширина колонок 140px относится к классу span2, который можно использовать в CSS описании документоа
Типографика — описания шрифтов, определение некоторых классов для шрифтов, таких как код, цитаты и т.д.
Медиа — предоставляет некоторое управление изображениями и видео.
